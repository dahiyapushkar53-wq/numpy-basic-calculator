## 📌 Overview

The **NumPy Basic Calculator** is a simple Python-based application designed to perform basic mathematical operations on user input.

The system takes numbers from the user and uses NumPy to quickly calculate important values like sum, average, maximum, and minimum.

---

## 🎯 Objective

To develop a simple and efficient tool that:

* Performs basic calculations using NumPy  
* Takes user input in an easy way  
* Helps understand NumPy fundamentals  

---

## 🚀 Features

* 📥 User input system  
* 🧠 Fast calculations using NumPy  
* 📊 Displays sum, average, max, and min  
* 🖥 Simple GUI using Tkinter  
* 🎨 Clean and easy interface  

---

## 🛠 Tech Stack

* **Python**  
* **NumPy**  
* **Tkinter (GUI)**  

---

## ⚙️ How It Works

1. User enters numbers (space-separated)  

2. System converts input into NumPy array  

3. Calculations performed:
   * Sum  
   * Average  
   * Maximum  
   * Minimum  

4. Output:
   * Results displayed in a popup window
